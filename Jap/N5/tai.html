欲しい、たい（願望（がんぼ）形）

1. 名詞（めいし）　＋　が　＋　欲（ほ）しい。

  • 私は、猫が欲しい。

  • お金が欲しいです。

  • 部屋に花が欲しいです。

2.　動詞（どうし）たい形

〜ます　→　〜たい

  • 買う　→　買いたい

  • 遊ぶ　→　遊びたい

  • 話す　→　話したい

  • 起きる　→　起きたい

  • 食べる　→　食べたい

  • 来る　→　きたい

  • する　→　したい

  • 見る　→　見たい

  • 会う　→　会いたい

  • 旅行する　→　旅行したい

3.　名詞　＋　が　＋　動詞たい形。

  • アニメが見たい。

  • おにぎりが食べたい。

  • 音楽（おんがく）が聴（き）きたい。

4. 否定形

〜たくない

  • お酒（さけ）が飲みたくない。

5. 過去形

〜たかった

6. 過去否定形

〜たくなかった

7. 三人称（さんにんしょう）：　たい　→ たがる（他動詞）

  • 彼女は画家（がか）になりたがる。

  • 娘はスポーツなら何でも好きです、スポーツ選手（せんしゅ）になりたかる。
