# 自動詞と他動詞

### 両方がある動詞

- 落（お）ちる　　＝　落（お）とす
- 沸（わ）く　　　＝　沸（わ）かす
- 燃（も）える　　＝　燃（も）やす
- 消（き）える　　＝　消（け）す
- 倒（たお）れる　＝　倒（たお）す
- 染（そ）まる　　＝　染（そ）める
- 重（かさ）なる　＝　重（かさ）ねる
- 見（み）つかる　＝　見（み）つける
- 曲（ま）がる　　＝　曲（ま）げる
- 乾（かわ）く　　＝　乾（かわ）かす
- 壊（こわ）れる　＝　壊（こわ）す
- 切（き）れる　　＝　切（き）る
- 起（お）きる　　＝　起（お）こす

### 無対自動詞

- 泳（およ）ぐ
- 死（し）ぬ

- 座（すわ）る

- 熟（じゅく）す
